osmtools предполагается набором утилит для работы с дампами OpenStreetMap

Основан на osmconvert/osmfilter.

В настоящий момент состоит из экспериментального порта osmupdate на python.

Цель портирования - заменить зависимость от wget и gzip на стандартную библиотеку python, 
поскольку она более кросплатформенна. Также предполагается снижение объема кода за счет,
например, использования готовых модулей разбора командной строки.
